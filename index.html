প্রপস (Props)

প্রপস (Props) হল একটি কম্পোনেন্ট থেকে অন্য কম্পোনেন্টে ডেটা পাস করার একটি উপায়। এটি কম্পোনেন্টকে ডাইনামিক এবং পুনরায় ব্যবহারযোগ্য (reusable) করে তোলে। প্রপস Read-only হয়, অর্থাৎ একবার পাঠানো হলে এটি চাইল্ড কম্পোনেন্টে পরিবর্তন করা যায় না।

const Greeting = (props) => {

স্টেট (State) এবং useState হুক

স্টেট (State) হল কম্পোনেন্টের নিজস্ব ডেটা যা সময়ের সাথে পরিবর্তন হতে পারে। যখন স্টেটের পরিবর্তন হয়, রিয়্যাক্ট কম্পোনেন্টটিকে পুনরায় রেন্ডার (re-render) করে। useState হুক ব্যবহার করে আমরা ফাংশনাল কম্পোনেন্টে স্টেট ব্যবহার করতে পারি। এটি একটি অ্যারে রিটার্ন করে যেখানে প্রথম এলিমেন্টটি বর্তমান স্টেট এবং দ্বিতীয়টি একটি ফাংশন যা স্টেট আপডেট করতে ব্যবহৃত হয়।

ইভেন্ট হ্যান্ডলার (Event Handlers)

রিয়্যাক্টে, আপনি HTML উপাদানের উপর বিভিন্ন ইভেন্ট (যেমন: ক্লিক, পরিবর্তন) হ্যান্ডেল করতে পারেন। onClick, onChange, onSubmit ইত্যাদি ইভেন্ট হ্যান্ডলার হিসেবে ব্যবহৃত হয়।

ক্লাসনেম (className) এবং স্টাইল (style)

className: এটি CSS ক্লাস যোগ করার জন্য ব্যবহৃত হয়। এটি HTML-এর class অ্যাট্রিবিউটের মতোই কাজ করে, তবে JSX-এ className লিখতে হয়।

style: এটি ইনলাইন CSS যোগ করার জন্য ব্যবহৃত হয়। এটি একটি জাভাস্ক্রিপ্ট অবজেক্ট গ্রহণ করে, যেখানে CSS প্রপার্টিগুলি ক্যামেলকেস (camelCase) ফরম্যাটে লেখা হয়।

useEffect হুক

useEffect হুক আপনাকে ফাংশনাল কম্পোনেন্টে সাইড ইফেক্ট (side effects) চালানোর সুবিধা দেয়। ডেটা ফেচিং, ম্যানুয়ালি DOM মডিফাই করা, বা টাইমারে কাজ করার জন্য এটি ব্যবহৃত হয়। এটি কম্পোনেন্ট রেন্ডার হওয়ার পর বা নির্দিষ্ট স্টেটের পরিবর্তন হলে কল হয়।

উদাহরণ কোড

import React, { useState, useEffect } from 'react';

const TitleChanger = () => {
  const [count, setCount] = useState(0);

  useEffect(() => {
    document.title = `আপনি ${count} বার ক্লিক করেছেন`;
  }, [count]); // ডিপেন্ডেন্সি অ্যারে

  return (
    <div>
      <p>গণনা: {count}</p>
      <button onClick={() => setCount(count + 1)}>ক্লিক করুন</button>
    </div>
  );
};
